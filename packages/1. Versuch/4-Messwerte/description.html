Messwerte

Aufgabe

Erstelle ein C-Programm, das vom Benutzer bis zu 50 Messwerte (Gleitpunktzahlen) einlesen kann. Im Anschluss sollen jeweils der größte und der kleinste Wert und der arithmetische Mittelwert dieser Messreihe ausgegeben werden.

Hinweis

In C Compact gibt es den Datentyp double nicht. Verwende float für die Messwerte.
Den arithmetischen Mittelwert (Durchschnitt) einer Zahlenmenge erhälst du, indem du alle Elemente addierst und das Ergebnis dann duch die Anzahl der Elemente dividierst. In diesem Fall würde die Formel für n = 15 so aussehen:


Eine Zahl kann dem Befehl readArray eingelesen werden. Dabei ist a das Array und n die Anzahl der Elemente des Arrays.
readArray(a, n);
Für diesen Befehl muss die Bibliothek array.h eingebunden werden.
#include <array.h>
