Bibliothek nicht gefunden

Was ist passiert?

Eine Bibliothek, die du verwenden willst, konnte nocht gefunden werden. Eine Bibliothek ist eine Datei, die mit #include <stdlib.h> eingebunden wird.

Fehlersuche

Standardbibliotheken enthalten viele der Befehle, die du aus C kennst und können mit folgendem Befehl eingebunden werden:

#include < stdlib.h >
Achte auf Tippfehler. Folgende Bibliotheken sind derzeit implementiert:
stdlib.h
stdio.h
math.h

Einbinden von eigenen Bibliotheken

Beim einbinden von eigenen Bibliotheken musst du beachten, dass sich diese im selben Ordner wie deine CMM-Datei befinden muss. Außerdem ist der Syntax etwas anders:

#include " yourlibrary.h "
