Integer Overflow

Was ist passiert?

Es wurde versucht einen Integer mit einer Zahl zu füllen wofür dieser nicht ausgelegt ist

Fehlersuche

Ein Overflow tritt auf wenn versucht wird eine Zahl in einen Datentypen zu schreiben, wofür dieser nicht ausgelegt ist.
Dies kann z.b. durch die Multiplikation von zwei großen Werten passieren.

Weiterführende Informationen

Ein Integer hat einen Wertebereich von -231 (-2147483648) bis 231-1 (+2147483647). Wenn dieser überschritten wird tritt ein Overflow auf.
