Vorwärtsdeklaration ohne Funktionsdeklaration

Was ist passiert?

Es gibt eine Vorwärtsdeklaration für eine Funktion, die Funktion selber ist aber nicht deklariert.

Beispiel

int foo();
...

Fehlersuche

Schaue dir die Vorwärtsdeklaration an, und deklariere entweder diese Funktion im Programm, oder lösche sie.
