Aufruf einer Funktion hat mehr Argumente als dessen Deklaration

Was ist passiert?

Du hast den aufruf einer Funktion mit zu wenigen Parametern gemacht

Beispiel

void foo();
...
foo(5);

Fehlersuche

Schaue dir die Deklaration der Funktion an, und ändere entweder die Funktion oder den Aufruf davon so ab, damit die Anzahl der Argumente übereinstimmt.
