Funktionsaufruf mit zu vielen Argumenten

Was ist passiert?

Du hast den Aufruf einer Funktion mit zu vielen Parametern gemacht.

Beispiel

void foo();
...
foo(5);

Fehlersuche

Schaue dir die Deklaration der Funktion an, und ändere entweder die Funktion oder den Aufruf davon so ab, damit die Anzahl der Argumente übereinstimmt.
