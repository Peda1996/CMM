schließende eckige Klammer vergessen?

Was ist passiert?

Du hast eine schließende eckige Klammer "]" vergessen.

Beispiel

void foo(int a[10]);

Fehlersuche

Der Compiler kann den Code nicht richtig verarbeiten, da er eine schließende eckige Klammer erwartet.
Schaue dir die gemeldete Stelle und den Code darüber an.
