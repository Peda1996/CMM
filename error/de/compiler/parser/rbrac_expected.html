Du hast eine schließende geschwungene Klammer "}" vergessen.

Fehlersuche

Der Compiler kann den Code nicht richtig verarbeiten, da er eine schließende geschwungene Klammer erwartet.
Dieser Fehler tritt auch auf wenn auf einen Array Index ohne Variable zugegriffen wird.
Schaue dir die gemeldete Stelle und den Code darüber an.
