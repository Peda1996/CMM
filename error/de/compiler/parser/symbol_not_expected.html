Unerwartetes Symbol

Was ist passiert?

Du hast ein Symbol im Quellcode plaziert welches an dieser Stelle nicht zugeordnet werden kann.

Beispiel

void foo;{}

Fehlersuche

Der Compiler kann den Code nicht richtig verarbeiten, da er dieses Symbol nicht erwartet.
Schaue dir die gemeldete Stelle und den Code darüber an, und achte besonders auf Zeichen die eigentlich nicht vorkommen sollten.
