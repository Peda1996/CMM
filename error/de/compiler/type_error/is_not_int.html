deklarierter Wert ist kein Integer

Was ist passiert?

Du hast versucht einen Integer mit einem Wert zu füllen welcher nicht korrekt ist.

Beispiel

int a = 0909;

Fehlersuche

Beachte dass C-Compact verschiedene Arten von Zahlenformaten unterstützt. Wenn diese Zahl mit einer 0 beginnt, wird diese als Oktalzahl ausgewertet.
Desweiteren besitzt ein Integer einen Wertebereich von -231 (-2147483648) bis 231-1 (+2147483647). Wenn dieser überschritten wird tritt ein Overflow auf.
