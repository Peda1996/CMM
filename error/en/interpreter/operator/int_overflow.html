float overflow

What happened?

You tried to fill a integer with a number which is to high or to low for this type.

Troubleshooting

An overflow can occour while performing a calculation which is result in a number which is outside the allowed range.
For example, that can happen if you multiply 2. big numbers.

Additional information

A floatingpoint number has a value range of -231 (-2147483648) to 231-1 (+2147483647). If this range is exceeded, an overflow occour.
