declared value is not an int

What happened?

You tried to fill an int with an incorrect value.

Example

int a = 0909;

Troubleshooting

Please note that C-Compact supports different types of number formats. If the number is starting with a 0, it is parsed as octal-number.
Furthermore, a int number has a value range of -231 (-2147483648) to 231-1 (+2147483647). If this range is exceeded, an overflow occour.
