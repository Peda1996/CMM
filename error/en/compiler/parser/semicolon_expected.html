Semikolon (or before) forgotten

Semikolon expected

What happened?

You have forgotten an semicolon ";".

Troubleshooting

Please notice that the compiler is detecting the missing semicolon in the next line.
For example, the error is detected in line 18, but the error is located in line 16:

15: int a = 5;
16: a = b * 7 
17:
18: printf("a = %i", a );

You should probably search the error in the line before reported line.
