Typing error?

What happened?

Your program is using unknow text.

Troubleshooting

Possible error causes

  • Wrong writing of a command e.g. fora
  • Wrong written datatype e.g. voidd
  • Typing error in variable names (look for uppercase and lowercase)
  • Typing error in function names
  • variable is not defined
Please notice that this error can be located sometimes one line before the error line.

Mostly this are typing errors.

Example

int abc = 6;
...
acb = 7;

or:

vvoid main(){
...
}
