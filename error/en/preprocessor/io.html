Error while reading the library

What happened?

A library which you would use cannot readed. A library is a file, which can be included with the command #include <stdlib.h>.

Troubleshooting

Check the the files inside the directory "clib" and user specific librarys. Probably the file is not readable or the program doesn't has read access to that file.
Additional Information are found on the error number "#2013" inside the list of internal Errors.
