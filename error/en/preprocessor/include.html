library not found

What happened?

A library which you would use is not found. A library is a file, which can be included with the command #include <stdlib.h>.

Troubleshooting

Standard libraries containing many function, which you know from C, and they can be included using the following way:

#include < stdlib.h >
Watch out for typing errors. The following librarys are implemented yet:
  • stdlib.h
  • stdio.h
  • math.h


Importing own libraries

If you include own librarys, you have to note that it is located inside the same directory as your .cmm file. Furthermore the syntax differs a little bit:

#include " yourlibrary.h "
